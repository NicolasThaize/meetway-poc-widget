🎯 Exemple - Tags de fonctionnalités personnalisés

Méthode 1: Configuration globale (config.js)

Modifiez le fichier config.js pour changer les tags par défaut :

// Dans config.js const MeetwayConfig = { // ... autres configurations ... featureTags: { tag1: '🚗 Covoiturage garanti', tag2: '💰 Économisez 15€', tag3: '🤝 Communauté bienveillante' } };

Méthode 2: Configuration locale (dans l'initialisation)

Passez les tags directement lors de l'initialisation du widget :

MeetwayWidget.init({ containerId: 'Meetway-widget-container', featureTags: { tag1: '🎫 Place garantie', tag2: '💸 Réduction 20€', tag3: '🌟 Expérience unique' } });
